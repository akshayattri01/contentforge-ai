# 🚀 ContentForge AI

ContentForge AI is a modern **AI-powered content generation web app** built using **React, Node.js, Express, and OpenRouter API**.

It helps users instantly generate high-quality content like:

✨ Blogs  
✨ Captions  
✨ Tweets  
✨ Marketing Copy  
✨ AI-generated text

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/akshayattri01/contentforge-ai.git
```

Move into project folder:

```bash
cd contentforge-ai
```

---

## 💻 Frontend Setup

Install dependencies:

```bash
npm install
```

Run frontend:

```bash
npm run dev
```

---

## 🔥 Backend Setup

Go to server folder:

```bash
cd server/server
```

Install dependencies:

```bash
npm install
```

Create a `.env` file and add:

```env
OPENROUTER_API_KEY=your_api_key
MONGO_URI=your_mongodb_uri
PORT=5000
```

Run backend:

```bash
node server.js
```

---

## 🔑 Environment Variables

The following environment variables are required:

